## 实现功能
* **矩阵键盘**
  * 外接4*4矩阵键盘，键盘上有0~9十个数字、加减乘除符号、等于、回退等十六个按键。
* **LCD1602**
  * LCD1602显示输入的数字和运算符号，并在按下等号并接收到串口信息后显示得数。
* **串口通信**
  * 执行机与终端机之间实现串口通信，通信过程为数据帧添加帧头帧尾以加强通信的稳定性。
* **执行计算**
  * 两个四位数之间的加减乘除，实现减法可以得出负数，乘法最大计算9999*9999且能显示得数，除法当除数为0时显示错误信息。
